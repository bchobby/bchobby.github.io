本用户使用centos8

1.安装lrzsz

yum install -y lrzsz

2.上传文件

输入

rz

选择官网下载下来的包进行上传

3.解压

tar -zxvf mysql-5.7.26-linux-glibc2.12-x86_64.tar.gz

4.移动并修改权限

mv mysql-5.7.26-linux-glibc2.12-x86_64 /usr/local/mysql
groupadd mysql
useradd -r -g mysql mysql

5.创建数据目录并赋权限

mkdir -p /data/mysql
chown mysql:mysql -R /data/mysql

6.配置my.cnf

vim my.cnf

[mysqld]
bind-address=0.0.0.0
port=3306
user=mysql
basedir=/usr/local/mysql
datadir=/data/mysql
socket=/tmp/mysql.sock
log-error=/data/mysql/mysql.err
pid-file=/data/mysql/mysql.pid
#character config
character_set_server=utf8mb4
symbolic-links=0
explicit_defaults_for_timestamp=true

7.初始化数据库

./mysqld --defaults-file=/usr/local/mysql/my.cnf --basedir=/usr/local/mysql --datadir=/data/mysql --user=mysql --initialize

8.修改root密码

查看root密码

cat /data/mysql/mysql.err


将mysql.server放置到/etc/init.d/mysql中

cp /usr/local/mysql/support-files/mysql.server /etc/init.d/mysql

启动

service mysql start

登录修改mysql密码

./mysql -u root -p

SQL指令

SET PASSWORD = PASSWORD('需要设置的密码');#设置密码
ALTER USER 'root'@'localhost' PASSWORD EXPIRE NEVER; #修改密码
FLUSH PRIVILEGES; #刷新

需要设置一下远程登录

use mysql;
update user set host='%' where user = 'root';
flush privileges;

解决linux下无法输入中文字符

解决一般情况下中文字符可能会替换成???

vi /etc/profile

添加如下：

export LANG="en_US.UTF-8"
export LC_ALL="en_US.UTF-8"
export LC_CTYPE="en_US.UTF-8"


source /etc/profile
