一、vim 仓库下载 vim 32 位 或者 64 位安装包]

安装入口

下载下来后双击运行，一路 next， 将其安装至非系统盘

将安装后的位置添加至环境变量中的 path 中，便于在cmd 或者 powershell 中命令行式的使用 vim

二、git 仓库下载 git

安装入口
同理将 git 安装好的路径添加至环境变量中的 path 变量中便于在命令行下使用 git 命令

三、将 .vimplus 克隆至用户文件夹

vimplus 仓库
刚进入cmd 时的位置即为用户文件夹：

"进入 cmd，不要变更位置，执行以下` git clone `命令(将 .vimplus 从仓库拷贝至你所处的文件夹)"
git clone https://github.com/chxuan/vimplus.git .vimplus

四、安装 vimplus

下载完成后，在 cmd 中进入 .vimplus 文件夹，执行以下命令（不需要 ./ ）

install.sh

五、移动个别文件以适配 vimplus 的设置

进入 .vimplus 文件夹，将 以下三个 vim 配置文件移动到用户文件夹下(与.vimplus 文件夹同一级别下)

将 .vimplus 下 autoload 文件夹中的 plug.vim 拷贝到 vim 安装路径下的 autoload 中

六、安装插件

cmd 命令行下执行：

" 1、打开 vim "
vim 
" 2、打开后输入以下命令安装插件 "  
,,i 
"（.vimplus/autoload/plug.vim 这个文件如果没有拷贝至 vim 安装路径中的 autoload ，
<leader><leader>i 的命令会被报错）"

七、扩展安装

powershell 下载入口
