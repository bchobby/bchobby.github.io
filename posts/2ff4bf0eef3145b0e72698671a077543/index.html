Ubuntu安装vnc并解决花屏问题

0. 环境

  • Ubutun 16.04.06 LTS

1.基本概念

VNC：VNC (Virtual Network Console)是虚拟网络控制台的缩写。它 是一款优秀的远程控制工具软件，由著名的 AT&T 的欧洲研究实验室开发的。

Ubuntu 存储库中还有几种不同的 VNC 服务器，如 TightVNC ， TigerVNC 和 x11vnc 。每个 VNC 服务器在速度和安全性方面都有不同的优点和缺点。

2. 安装设置vnc4server

step1. 安装vnc4server

apt-get install vnc4server

step2. 设置密码

输入vncserver，初次启动的时候需要设置初始密码，密码最少6位。

root@x60s:~# vncserver

You will require a password to access your desktops.
password:
Verify:

New 'X' desktop is x60s:1

Starting applications specified in /root/.vnc/xstartup
Log file is /root/.vnc/x60s:1.log

step3. 修改xstartup文件

文件在：~\.vnc\中，一般初始化是没有的，需要你自己创建。

  • 如果是root环境，文件：\root\.vnc\xstartup
  • 如果是其他用户环境，比如用户arthur，文件：\home\arthur\.vnc\xstartup
vim ~\.vnc\xstartup
# 记住编辑完了之后，要提权，原先花屏的原因不排除是因为这个文件没有提权
sudo chmod +x ~\.vnc\xstartup

xstartup文件内容如下：

#!/bin/sh
# Uncomment the following two lines for normal desktop:
export XKL_XMODMAP_DISABLE=1
unset SESSION_MANAGER
# exec /etc/X11/xinit/xinitrc
unset DBUS_SESSION_BUS_ADDRESS
gnome-panel &
gnmoe-settings-daemon &
metacity &
nautilus &
gnome-terminal &

step4. 安装gnome

apt-get install gnome-panel gnome-settings-daemon metacity nautilus gnome-terminal

step5. 重启系统，气动vnc服务

重启ubuntu操作系统，启动vncserver服务。

  • 重启命令： reboot
  • vnc服务启动：vncserver

step6. windows终端访问

用ip:5901，输入设置的密码就能进行访问了。

3. 问题汇总

有时候（大多时候）会出现花屏，鼠标是一个x

解决1

有时候花屏是因为没有对远端Ubuntu进行桌面共享设置

解决2

安装gnome

apt-get install --no-install-recommends ubuntu-desktop gnome-panel gnome-settings-daemon metacity nautilus gnome-terminal -y

安装dpi

sudo apt-get install xfonts-100dpi
sudo apt-get install xfonts-75dpi

4. vncsever命令

VNC的启动/停止/重启
service vncserver start/stop/restart
关闭具体的vncserver命令:

  • vncserver -kill :1
  • vncserver -kill :2

设置密码
vncpasswd

OK！
