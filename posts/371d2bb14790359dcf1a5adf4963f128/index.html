2.安装和引入vant

vant官网

# 安装
vue2.x：npm i vant@latest-v2 -S
vue3：npm i vant -S

# 引入——按需引入（按需引入之后就不能再进行全局引入）
npm i babel-plugin-import -D
// 在.babelrc 中添加配置
{
  "plugins": [
    ["import", {
      "libraryName": "vant",
      "libraryDirectory": "es",
      "style": true
    }]
  ]
}
