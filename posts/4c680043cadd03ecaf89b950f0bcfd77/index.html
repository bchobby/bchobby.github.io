redhat中Telnet服务器搭建

主要说明：

利用yum源来下载安装Telnet服务器

准备步骤：进入root界面

[ly@localhost 桌面]$ su  

密码：

[root@localhost 桌面]#

  1. 配置好yum源文件（此处省）
  2. 检查是否安装了Telnet和xinetd
  3. 开始进行安装

安装Telnet服务端

#yum -y intall telnet-server.86_64

 

注：一般如下找不到镜像文件的原因是 没有连接网络！！！

此时需要在本电脑连好网络后，再在虚拟机桌面右上角出开启连网，如图

安装Telnet客户端

#yum -y install telnet.x86_64

4、安装xinetd进程

 注：xinetd进程是Telnet的守护进程

#yum -y install xinetd.x86_64

5、检查是否都安装成功

# rpm -qa | grep telnet   

 

 # rpm -qa | xinetd

6、 安装成功后，将xinetd服务加入开机自启动：

# systemctl enable xinetd.service   

如果失败了，一般是没有关闭防火墙造成，需要暂时关闭防火墙：

# systemctl stop firewalld

7、 将telnet服务加入开机自启动：

# systemctl enable telnet.socket

8、启动telnet和xinetd两个服务即可

# systemctl start telnet.socket

# systemctl start xinetd

9、 开启端口

注：安装telnet服务后，默认23号端口会被封住，以下方法

法一：开启端口     法二：关闭防火墙

查看是否开启了23号端口

# firewall-cmd --query-port=23/tcp  

显示no没有开启，下面进行开启

# firewall-cmd --zone=public --add-port=23/tcp --permanent

 重新加载firewall-cmd

# firewall-cmd --complete-reload  

重新查看23号端口是否开放

# firewall-cmd --query-port=23/tcp

现在就可以使用telnet命令进行远程登录了

10、测试telnet连接

首先要开启telnet服务，默认是关闭的

步骤：真机（这里说的是window本机，非虚拟机）【控制面板】->【程序】->【启动或关闭windows功能】->勾【Telnet Client】->【关闭】

然后就可以在window终端cmd进行测试了

如下命令：

telnet +ip（真机IP地址）   即可

然后输入虚拟机用户名和密码就可远程登录
