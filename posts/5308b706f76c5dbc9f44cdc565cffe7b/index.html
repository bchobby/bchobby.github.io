  • 由于 R n \R^{n} Rn内最多可以有 n n n个向量组成线性无关组，因此上述定义满足 0 ≤ k ≤ n 0\leq k \leq n 0≤k≤n。这表明：在有限维向量空间内，单纯形不能无限扩张， R n R^{n} Rn空间中最多存在 n n n-单纯形
