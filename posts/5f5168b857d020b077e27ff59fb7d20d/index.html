总结下在离开教育网后公网各种接入IPV6的方法

1. 最简单的，，也是我强烈推荐的。不管你是不是使用路由器拨号（NAT用户）还是不经过路由器直连的ADSL用户，都适用，就是Tunnel broker。

是 gogo6 家的产品，需要注册后才能下载和使用，但是使用是免费的，基本没啥限制，速度很ok，我记得这个默认的服务器 anonymous.freenet6.net 没有被方校长认证。自己刚刚毕业的时候除了交大隧道就是靠他活了。 用这个连CG，无锡电信延迟基本在400-500ms，虽然有点高，但是访问网页没有想象中的那样卡，可以接受。下载基本也可以把我家2M小水管盛满。
 
 
除了默认的服务器，你可以试试其他你们tunnel broker隧道服务器， 比如亚太电信的 。这个你们可以放Google搜索。很多的。

2 . 如果你不使用路由器通过ADSL拨号直连，而且你是中国电信和中国联通这种主流电信运营商的用户（FTTB小区宽带有可能例外），基本上你ADSL拨号博取的就是公网IP。确认办法就是你拨号后，查看ADSL拨号连接的状态，看下IP地址是不是和你在  http://www.123cha.com  看到的一致，如果一致，基本就是公网用户了，这个时候你可以考虑使用上海交大的公网isatap隧道。（广电，移动等小众运营商用户洗洗睡吧）
使用方法和普通的isatap隧道是一样的。打开开始菜单->管理员权限运行cmd命令提示符（记得关闭Windows防火墙）
（XP用户第一次使用记得先运行ipv6 install）
  1. netsh interface 6to4 set stat dis
  2. netsh interface teredo set stat dis
  3. sc config iphlpsvc start= auto
  4. net start iphlpsvc
  5. netsh interface ipv6 isatap set router isatap.sjtu.edu.cn
  6. netsh interface ipv6 isatap set state dis
  7. netsh interface ipv6 isatap set state en

记得 DNS最好不要换成电信的。弄成8.8.8.8。原因看文末介绍的连接。
然后 ping -6 ipv6.cgbt.cn 如果提示域名无法解析，ping -6 2001:da8:205::100 如果后者通，要么你换dns，要么绑定host。看文末介绍的连接。

3 .如果你在路由器或者其他NAT网关后面，比如某些使用FTTB、FTTH的的小区宽带 （D大说通俗的讲就是如果你是多机通过路由器上网(路由器拨号)）
除了方法1里面的tunnel broker客户端，你可以考虑使用teredo隧道。不过方校长可能不会让你用的很舒服，而且teredo隧道存在的情况下，可能会让某些流量走teredo的ipv6。 这就是为啥有些家里用路由器的同学也能上ipv6但是速度很慢的原因。
建议大家禁用它。
  1. netsh interface teredo set stat dis
