如需转载，请注明转载自：blog.csdn.net/it1988888

方案一--下载颜色主题配置文件，离线导入配色方案的设置：

1. 下载颜色配置文件：

http://www.eclipsecolorthemes.org

推荐选择配色方案：Tomorrow Night - by Chris Kempson - Website

Eclipse Color Theme (XML) - for Eclipse Color Theme Plugin

Eclipse Preferences (EPF) - for Eclipse Import

推荐下载".EPF"文件格式，可以离线导入Eclipse软件设置。

 

2.导入颜色配置文件至Eclipse:

Eclipse -> File -> Import -> General -> Preferences -> (定位至已经下载的“.EPF”文件) -> Finish.

即可生效。

Eclipse -> Help -> Install New Software -> Add->Archive ->

-> 安装。

推荐“NightLion Aptana Theme”
点击Apply -> OK即可。

安装主题插件来设置Eclipse软件框架的配色方案：

如需转载，请注明转载自：blog.csdn.net/it1988888

1. 确保你的Eclipse版本高于Juno版（即Eclipse 4.2）,更早的版本可能不支持;

2. 下载Dark Juno主题插件安装包：

https://github.com/downloads/rogerdudler/eclipse-ui-themes/com.github.eclipsecolortheme.themes_1.0.0.201207121019.zip

Dark Juno主题插件可以修改Eclipse软件框架、视图、工具栏的颜色。注意：该插件不能设置文本编辑区域（Editor）的颜色，需要通过另行导入颜色主题配置文件的方式，来设置Editor区域的颜色（参考上文）。

3.解压主题包“com.github.eclipsecolortheme.themes_1.0.0.201207121019.zip”

至Eclipse安装目录的dropins子目录下，重启Eclipse。

4. 选择配色主题，步骤如下：
Eclipse -> Window -> Preferences -> General -> Appearance -> Theme -> Dark Juno主题，

即可。

Eclipse的背景设置（黑色系）

转载自：https://www.cnblogs.com/hjy97/p/10134333.html

一直都在用eclipse写代码，原来还能受得了刺眼的白色背景，随着时间推移慢慢地感觉眼睛受不了这种刺激，遂决定换成一个黑色背景。

上网搜了改变背景的方法，大多是换成自带的dark主题或者Color Theme的一系列主题。但是我觉得dark主题太丑，而Color Theme主题只能应用于代码编辑区，别的区域依旧是白色背景，即使我先应用dark，然后应用Color Theme中的主题使eclipse变成黑色，代码编辑区也是想要的效果，但是一旦重新打开eclipse，主题会自动变回dark，在尝试了众多方法后，找到了一种有效方法。

步骤如下：

Step1:下载com.github.eclipseuitheme.moonrise_0.8.9.jar放入eclipse的plugins文件夹中

Step2:打开eclipse，window->preferences->General,点击Appearance，选择Theme为MoonRise(standalone)，应用。这一步保证了整个eclipse区域为黑色背景，想要在黑色背景前提下更换其他主题的转至step3

Step3:点击window->preferences->General->Appearance->Color Theme,选择一款你中意的代码编辑区主题，这里我选择的是Sublime Text2，应用，这样就可以享受黑色背景高亮代码的eclipse了。

如果没有Color Theme，可以点击Help->Eclipse Marketplace->Search->find查找Color Theme，安装Eclipse Color Theme1.0.0
