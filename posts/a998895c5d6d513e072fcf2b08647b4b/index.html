16.接下来该写实现流水灯的代码了，这里用到的是 Keil uVision5 。

17.双击Keil uVision5进入。

18.新建工程：打开keil → project → New μVision Project

 

 19.输入工程名称，点击保存，新建成功。

 20.新建空白文档。

 

 21.将下面的代码写进文档。

#include<reg52.h>
void delay(unsigned int i);
 
int main(void)
{
	while(1)
	{
		
		P2=0xfe;      //1111 1110 
		delay(50000);
		P2=0xfd;      //1111 1101
		delay(50000);
		P2=0xfb;      //1111 1011
		delay(50000);
		P2=0xf7;      //1111 0111
		delay(50000);
		P2=0xef;      //1110 1111
		delay(50000);
		P2=0xdf;      //1101 1111
		delay(50000);
		P2=0xbf;      //1011 1111
		delay(50000);
		P2=0x7f;      //0111 1111
		delay(50000);

	}
}

void delay(unsigned int i) //延时函数
{
	while(i--);	
}

21.保存为.c文件

 22.将新建的Main.c文件添加到Source Group 1 文件夹下。
