【Git】error: RPC failed； curl 92 HTTP/2 stream 0 was not closed cleanly: PROTOCOL_ERROR


一、问题


git提交报错：error: RPC failed; curl 92 HTTP/2 stream 0 was not closed cleanly: PROTOCOL_ERROR (err 1)


二、原因


http2本身的bug

三、解决


1、方法一：直接换掉Git的http版本

git config --global http.version HTTP/1.1

2、方法二：更改Git的http克隆为ssh，使用ssh进行提交和拉取代码
