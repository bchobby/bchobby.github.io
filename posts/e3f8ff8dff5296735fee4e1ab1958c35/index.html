四路服务器 虚拟机 推荐整合比,三套服务器虚拟化整合方案特色对比

【IT168 专稿】目前服务器领域，哪些技术或产品正在流行？虚拟化和云计算是当仁不让的焦点了。服务器虚拟化技术之所以如此流行，是因为它能够帮助用户降低成本，同时也可起到简化对资源以及对资源管理访问的目的。目前主流服务器厂商，针对这一应用领域均有自己的相应产品或整体化解决方案，一些企业也将虚拟化服务器产品的采购提上日程，因此这里总结和对比一下目前市面上较受关注的虚拟化服务器，以期能给有需求的用户一个采购参考。

一、IBM System x 服务器虚拟化整合方案

方案亮点：IBM System x 高端服务器+ VMware 虚拟化数据中心整体打包方案，产品组成如图1所示。

图1 IBM整合方案示意

方案配置：

服务器：IBM System x3850 M2 7141I02一台。

虚拟化软件：VMware VI3 Standard 版本(含双 CPU 授权) 一套。

存储：IBM Storage DS3200 SAS 磁盘阵列一套(含 4 x 146GB 15k 磁盘及相关配件)。

方案说明：IBM自2001年就和VMware具有强有力的合作关系，包括专门的VMware 虚拟化平台的ServerProven服务器兼容性测试，以及整合了虚拟机的高级系统管理功能等。这套针对普通企业环境的虚拟化整合方案，具有出色的性能表现和较高的性价比优势。因为IBM System X3系列服务器提供了更高的内容容量配置，而内存容量在虚拟化环境中是非常重要的，而且X3架构的"虚拟四级缓存(XceL4v)"也提高了VMware的总体性能。

方案点评：IBM的这套服务器虚拟化整合方案，旨在为普通中小企业打造最简单的虚拟化环境，配置的IBM System x服务器是IBM非常成熟的一个系列，可以很好的保障整个系统的可用性；同时其多种高级内存保护机制、光通路诊断、冗余部件和预测故障分析等特性，又可有效保障关键业务应用。

二、戴尔预配置型虚拟化套餐

方案亮点：与微软新型虚拟化技术的整合与支持，有效提升了整体环境的通用性和易用性，方案产品组成如图2所示。

图2 戴尔整合方案示意

方案配置：

服务器：R710服务器。

虚拟化软件：微软虚拟化套件(包括内含Hyper-V的Windows Server 2008和System Center Essentials/ Virtual Machine Manager 2008)。

存储：PowerVault MD3000i和DL2000存储。

其他：PowerConnect网络交换机。

方案说明：戴尔推出的这套端到端的虚拟化方案，最大的特色就是配套产品性能强劲，而且整体组合下来的成本性价比较高。比如，戴尔R710这款服务器，支持双路至强5500处理器，支持144GB DDR3内存，性能和规格提升到了新的层次；同时，随服务器一同提供的微软系统中心虚拟机管理软件，可提供在多厂商环境中对其他虚拟化程序的统一管理，这样简化系统管理的方式可帮助客户提高效率，最大化利用资源。另外，戴尔特有的模块化虚拟化咨询服务，通过WEB调查、最佳实践、对比数据和参考架构等，帮助用户对自己的虚拟基础架构进行设计、管理和优化。

方案点评：戴尔的这套虚拟化整合方案，包括服务器、存储模块和软件在内的产品都可以预先配置，这样就为缺乏一定的服务器配置知识的中小企业客户，以及需要快速完成服务器虚拟化和应用过程的客户，提供了很大的选购空间。

三、浪潮双机热备虚拟整合方案

方案亮点：把虚拟化技术与安全措施结合，使用户在大幅提高投资回报率的同时，得到更高的可靠性。方案示意如图3所示。

图3 浪潮整合方案示意

方案配置：

服务器：浪潮NX660D 刀片服务器、浪潮四路服务器NF560D2与NF520D2。

虚拟化软件：浪潮虚拟主机管理软件、Virtuozzo企业虚拟化软件。

存储：浪潮 As300Storage共享存储。

方案说明：本方案简单而言，就是利用虚拟化技术在一台物理机器上创建多个虚拟机，与多个运转不同应用的物理机实现双机热备，从而达到一对多的系统安全热备方案，同时亦可实现对创建在不同物理机上的虚拟机之间的虚拟机热备方案；再通过外置的数据共享存储来实现业务的连续性；其他创建的虚拟机分别执行不同的任务。

值得一提的是采用的浪潮As300共享存储设备，它自带RAID 控制器，能实现RAID0，1，5 等RAID 级别，用户可以选择适当的级别RAID 提高共享存储中数据的可用性和性能。

方案点评：采用虚拟机跟物理机的双机方案，由于一台物理机器上可以安装多个虚拟机，不但充分利用了现有资源，而且从某种意义上来说降低了成本，并且也能达到实际环境的要求，做到最小成本的无间断服务。

四、对比总结

以上三种虚拟化整合方案，均针对中小企业用户而定制，功能特点都比较突出；用户在选购时其实要考虑的根本是如何节省成本和方便管理。面对虚拟化，我们只有作出正确的评估，选择正确的软硬件，以正确的方式推进，才能规避虚拟化过程的风险，最终顺利实现虚拟化部署。

另外，从现在谈的云计算的概念来看，其实云计算也是虚拟化的一种实现模式，有需求的企业用户也可考虑选择具备"云"概念的服务器产品，来构建企业虚拟化环境。
